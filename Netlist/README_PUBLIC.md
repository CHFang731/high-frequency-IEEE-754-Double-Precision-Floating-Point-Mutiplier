# Netlist Public Notes

The public copy keeps the gate/post-layout test benches, run files, and public
timing constraint file so the verification intent is visible.

The run files keep the original referenced standard-cell model filename for
context, but the private model file itself is not included. Synthesized gate
netlists, post-layout netlists, SDF, SPEF, GDS, and other physical
implementation outputs are also not included.
