# Innovus APR Public Notes

This directory keeps the original APR step filenames and the general Cadence
Innovus command flow. Generic APR commands are not treated as confidential.

Only foundry/PDK/library-specific values were removed or replaced with
`<PRIVATE_...>` placeholders. Redacted content includes process setup values,
tap/endcap/filler/tie cell names, routing layer constraints, metal
width/spacing values, GDS stream-out map paths, physical-library GDS references,
and operating-corner values. Refill those placeholders with authorized
technology collateral before running APR.
