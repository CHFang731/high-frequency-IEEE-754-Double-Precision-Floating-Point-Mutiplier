# Synthesis Public Notes

`SYN_RTL.sdc` is kept because it only describes the public 0.5 ns clock and I/O
timing assumptions.

`SYN_ADFP_RC.tcl` keeps the Cadence Genus synthesis command flow used by the
project. Generic synthesis commands are not treated as confidential.

Only technology-collateral filenames are redacted with `<PRIVATE_...>`
placeholders, including the standard-cell timing library, technology LEF,
standard-cell LEF, and QRC technology file.
